## 🚀 Overview

**PDF to Image Converter** is a simple and fast web application that allows users to convert PDF files into high-quality images directly in the browser.

This tool is built using **pure frontend technologies**, so no backend or server is required. Everything runs locally in your browser, ensuring speed and privacy.

---

## ✨ Features

* 📤 Upload PDF files (drag & drop supported)
* 🖼️ Convert each PDF page into images
* 📸 High-quality image output
* 📚 Preview all pages like a gallery
* 💾 Download images individually or all at once
* ⚡ Fast and lightweight (no backend needed)
* 🔒 100% client-side (your files stay private)

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* PDF processing libraries (like PDF.js)

---

## 📂 How It Works

1. Upload a PDF file
2. The app reads the file in your browser
3. Each page is rendered as an image
4. You can preview and download images

---

## 📦 Installation (Run Locally)

```bash
# Clone the repository
git clone https://github.com/your-username/pdf-to-image-converter.git

# Open the project folder
cd pdf-to-image-converter

# Run by opening index.html in browser
```

## 🌐 Usage

* Open the live demo or run locally
* Upload your PDF
